# Dependencies

* A C++ compiler supporting C++ 11
* Boost
* CuteR
* CMake >= 3.6
* py2exe

# Build

This program should be built with CMake and you need to configure Boost path and python executable path first.
```
mkdir build
cd build
cmake ..
cmake --build .
```
